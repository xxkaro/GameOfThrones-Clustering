# README

## Project: Clustering Game of Thrones Characters

### Description
This project focused on clustering characters from the Game of Thrones universe based on their house, culture, popularity, age and other related attributes. The aim was to identify and group characters with similar traits and backgrounds.

### Dataset
The dataset used for this project was sourced from [Kaggle]([https://www.kaggle.com](https://www.kaggle.com/datasets/mylesoneill/game-of-thrones/data?select=character-predictions.csv)).
